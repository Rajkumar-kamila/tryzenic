Qu'est-ce qu'un plan de taggage ?<\/h2>\n\n\n\n

Dans une strat\u00e9gie mobile marketing, le plan de taggage<\/strong> est un document qui liste un certain nombre d'\u00e9v\u00e9nements pr\u00e9d\u00e9finis li\u00e9s \u00e0 une action dans une application. Ce document, aussi appel\u00e9 plan de tracking, est utile pour optimiser la collecte de donn\u00e9es et le suivi de performance. Il permet en effet de mettre en place des KPI \u00e0 suivre en fonction des actions men\u00e9es par les utilisateurs dans l'application.<\/p>\n\n\n\n

Au-del\u00e0 de cette fonction analytique qui aide \u00e0 booster la performance, le plan de taggage permet de configurer l'envoi automatique de notification selon des actions pr\u00e9cises :<\/p>\n\n\n\n

    \n
  • Ouverture ou fermeture de l'app,<\/li>\n\n\n\n
  • Inscription ou connexion \u00e0 un compte,<\/li>\n\n\n\n
  • Clic sur un bouton d'achat,<\/li>\n\n\n\n
  • Abandon de panier<\/li>\n\n\n\n
  • Non utilisation de service apr\u00e8s X temps<\/li>\n\n\n\n
  • R\u00e9ception d'un message personnel<\/li>\n\n\n\n
  • Rappel<\/li>\n\n\n\n
  • Relance\u2026<\/li>\n<\/ul>\n\n\n\n

    On retrouve ces notifications dans des applications notamment (notification in-app) mais \u00e9galement sur des sites web, par email ect... Le plan de taggage permet de cr\u00e9er un parcours visant \u00e0 interpeler l'utilisateur avec un message (notification, email, popup etc...). <\/p>\n\n\n\n

    Dans un plan de taggage, on d\u00e9termine les objectifs, les donn\u00e9es \u00e0 traquer ainsi que les cons\u00e9quences, qui sont souvent une interaction avec l'utilisateur. Dans une application mobile par exemple, le plan de taggage et le plan de notification sont particuli\u00e8rement li\u00e9s<\/p>\n\n\n\n

    L'utilisation du plan de taggage<\/strong> vise principalement \u00e0 am\u00e9liorer le taux d'engagement<\/strong>. Il se construit \u00e0 l'aide d'\u00e9tudes : on \u00e9tudie le produit et les besoins en termes de relance, mais \u00e9galement l'utilisation du service par le consommateur (\u00e0 l'aide de traqueurs) afin de d\u00e9celer les meilleurs moments pour envoyer un message. <\/p>\n\n\n\n
