L'h\u00e9bergement web : d\u00e9finition<\/h2>\n\n\n\n

Toute mise en ligne d'un site sur internet commence par l'h\u00e9bergement web. Il s'agit d'un ou de plusieurs serveurs<\/strong> dans lesquels tous les fichiers qui constituent le site sont h\u00e9berg\u00e9s. Pour mieux cerner le sujet, il faut comprendre comment fonctionne le web. Quand vous naviguez sur internet, votre ordinateur envoie des requ\u00eates \u00e0 un serveur. Celui-ci va ensuite chercher les fichiers concernant le site demand\u00e9 dans sa base de donn\u00e9es. Le serveur renvoie enfin les informations trouv\u00e9es vers votre ordinateur pour afficher la page web.<\/p>\n\n\n\n

Par d\u00e9finition, un h\u00e9bergeur est l'entreprise qui met \u00e0 disposition ses serveurs pour que les professionnels et les particuliers puissent mettre en ligne leurs sites web. Auparavant, chacun utilisait son propre ordinateur comme serveur pour h\u00e9berger son site. On a vite compris qu'une fois que le nombre de visiteurs augmente, un simple ordinateur n'est plus assez puissant pour g\u00e9rer simultan\u00e9ment la r\u00e9ception des requ\u00eates et l'envoi des r\u00e9ponses. C'est l\u00e0 qu'entrent en jeu les h\u00e9bergeurs web. Les serveurs propos\u00e9s par ces derniers disposent de processeurs puissants, d'un large espace de stockage et des m\u00e9moires vives de grande capacit\u00e9<\/strong>. Ces caract\u00e9ristiques sont importantes pour la rapidit\u00e9 de chargement des sites web qui y sont h\u00e9berg\u00e9s.<\/p>\n\n\n\n
