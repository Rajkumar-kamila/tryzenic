jQuery(function(){\n    var current = window.location.href;\n    jQuery('.MA-CLASSE').each(function(){\n        var $this = jQuery(this);\n        if($this.attr('href') === current){\n            $this.addClass('active');\n        }\n    });\n});\n\/\/Changer .MA-CLASSE par la classe du lien \u00e0 Tagger\n\/\/Ajouter .MA-CLASSE.active pour personnaliser le lien actuel<\/code><\/pre>\n\n\n\n
Ce snippet jQuery a pour fonction de parcourir toutes les balises HTML ayant la classe 'MA-CLASSE' sur une page web. Il v\u00e9rifie ensuite si l'attribut 'href' de chaque balise correspond \u00e0 l'URL actuelle de la fen\u00eatre du navigateur (stock\u00e9e dans la variable 'current').<\/p>\n\n\n\n
Si l'attribut 'href' d'une balise 'MA-CLASSE' correspond \u00e0 l'URL actuelle, alors cette balise se voit ajouter la classe 'active'.<\/p>\n\n\n\n
En r\u00e9sum\u00e9, ce code permet de marquer comme 'active' les liens qui pointent vers la page actuellement affich\u00e9e dans le navigateur.
