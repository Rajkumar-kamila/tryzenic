Qu\u2019est ce que le Mobile First et pourquoi est-il important ? <\/h2>\n\n\n\n

Il y a encore quelques ann\u00e9es, on avait l'habitude d'adapter d\u2019abord le web design \u00e0 l\u2019ordinateur, avant de progressivement l\u2019optimiser pour le mobile. Aujourd\u2019hui, la tendance s\u2019est largement invers\u00e9e. Et pour cause, en 2020, 2\/3 du trafic en ligne mondial provenait des smartphones, soit une augmentation de 16% par rapport \u00e0 2019, selon une \u00e9tude ContentSquare<\/a>. <\/strong><\/p>\n\n\n\n

En 2022, on imagine facilement que ce chiffre ait encore grimp\u00e9. Le Mobile First Design n\u2019est donc plus un march\u00e9 de niche mais bien un march\u00e9 qu\u2019il est essentiel de prendre en compte dans sa strat\u00e9gie digitale. Et cette inversion de la tendance impose aux d\u00e9veloppeurs web de nouveaux usages. <\/p>\n\n\n

Mobile first ou Desktop first ?<\/h2>

Google recommande le mobile first, mais analysez bien les visites de votre site pour savoir si votre audience le consulte sur davantage sur mobile ou sur desktop. Une analyse compl\u00e8te permet de faire le choix du mobile first.<\/p><\/div><\/div><\/div>\n\n\n

Aujourd\u2019hui, faut-il toujours privil\u00e9gier l\u2019approche mobile first ? <\/h3>\n\n\n\n

Il est tr\u00e8s pertinent de s\u2019assurer que notre site web s\u2019adapte parfaitement au dernier t\u00e9l\u00e9phone sorti. Cependant, il peut \u00eatre int\u00e9ressant de consulter, en premier lieu, les donn\u00e9es analytiques de notre site avant d\u2019envisager l\u2019approche Mobile First ou au contraire Desktop first. <\/p>\n\n\n\n

Le mobile First est une approche de construction diff\u00e9rente, qui permet d'optimiser un site sur mobile, quitte \u00e0 prioriser le mobile au desktop (PC \/ MAC). Il ne faut pas tenir pour acquis la tendance du mobile first<\/strong>: Chaque produit d'adresse \u00e0 une cible bien particuli\u00e8re. Sur le site de l'agence de communication digitale<\/a> Citron Noir par exemple, nous n'avons pas adopt\u00e9 le mobile first. En effet, les statistiques du site web sur l'ann\u00e9e pr\u00e9c\u00e9dente montre que les 2\/3 des visiteurs du site sont sur desktop.<\/p>\n\n\n\n
